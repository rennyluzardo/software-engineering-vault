## 🚀 Featured Case Studies

### ⚡ [High-Performance Bundle Strategy](./high-performance-bundle-strategy/)
**Problem:** A monolithic enterprise platform with a >2MB initial bundle size affecting Core Web Vitals.
**Solution:** Implemented a granular code-splitting strategy, tree-shaking optimization, and a dynamic icon registry.
**Impact:** Projected reduction of **60% in initial load time** and a **+40 point increase** in Lighthouse performance scores.

### 🎨 [Gestalt UI & Accessible Dark Mode](./gestalt-ui-dark-mode/)
**Problem:** High cognitive load and visual fatigue in data-dense administrative dashboards.
**Solution:** Engineered a theme system based on **Gestalt Psychology principles** (Figure-Ground, Continuity) and WCAG 2.1 contrast standards.
**Impact:** Improved accessibility (6.5:1 contrast ratios) and reduced eye strain for power users.
